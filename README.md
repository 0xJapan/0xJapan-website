0xjapan.org
-----------

https://0xjapan.org

0xjapan のウェブサイト。 

### ローカル環境で編集する

リポジトリをクローンする。

```sh
git clone git@github.com:0xJapan/0xjapan.github.io.git
```

依存する gem をインストールし、開発用サーバーを立ち上げる。

```sh
cd 0xjapan.github.io
bundle install
bundle exec jekyll serve
```

ブラウザで http://localhost:4000/ を開く
